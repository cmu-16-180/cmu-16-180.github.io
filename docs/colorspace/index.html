Color Space Notes

  • RGB channels are shown using standard additive lighting (Black = 0, Bright Color = 255) to reflect how camera sensors capture light.
  • HSV (Hue, Saturation, Value) aligns closer to human perception.
  • Subsampling is applied automatically to 3D plots to maintain browser performance.
